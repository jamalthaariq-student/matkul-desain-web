Berkenalan dengan Fedora Silverblue

Dipublikasikan pada: 16 Januari 2023

Sebagai mahasiswa IT, saya tahu betul betapa pentingnya tools dan libraries dalam menyelesaikan tugas dan proyek kuliah saya. Demi tugas kuliah, saya harus menginstall berbagai macam software seperti Python dengan pip sebagai package manager-nya, Java, PHP, NodeJS, berbagai macam database seperti MySQL, Postgresql, Mariadb, dan bahkan virtual machine! Namun, beberapa tools itu hanya saya gunakan selama menempuh mata kuliah yang membutuhkan saja. Setelah saya menyelesaikan mata kuliah tersebut, saya tidak menggunakan tools dan libraries itu lagi. Hal ini dapat menyebabkan komputer saya menjadi berat dan lambat, atau istilahnya bloated.
